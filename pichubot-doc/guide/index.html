# go-Pichubot

基于 Onebot v11 (opens new window) 协议，使用 golang 开发的机器人框架，目前支持正向Websocket通信。

后续将实现更多拓展功能。

已在 Github go-Pichubot (opens new window) 开源。

# 提交 issue

在您的使用过程中如有发现bug，请前往github仓库提交issue

请注意：提交issue需要

  • 附上报错的日志文件（如果是错误）
  • 直截了当的指出重点
  • 按照Template提交

注意

开发者并没有义务回复您的问题, 您应该具备基本的提问技巧。
